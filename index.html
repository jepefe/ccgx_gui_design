Latest changes

  • Added icons.
    Users can now easily see the time, the GPS connectivity, alert status and VRM Portal Status
  • Changed "Settings > Alarms" to "Alarm Settings".
    This way the menu goes less deep and users have to do less actions to get there.
  • Changed button structure.
    The overview can now be opened with the top left button. A user can always return to the main menu with the top right button. A user can navigate and change settings with the middle pad. The circle button is the "select" button.
  • Changed select (radio) form controls to show value and made it a radio button.
    This way users can view the value without looking and if they want to change it they will know that only one item can be selected.
  • Added icon to show that more items are visible if you scroll down.
    This way users know that more items are below the fold.
  • Changed "Setup" to "Settings".
    It adds consistency if all menu items have the same name.
  • Moved the firmware version and build date to the "Firmware" tab.
    This way the information is more organized and the "General Settings" tab looks cleaner.
  • Toggles have text to further clarify the option.
    For example: "Alarm relay on" was confusing. You turned the toggle on or off but it would still say: "Alarm relay on". This way it says it in the toggle and you can change the text to "Alarm relay", same for "Logging enabled" in the VRM Online Portal settings.
  • Created General Information menu item for specific products.
    This way there is an overview of menu items per product, and users don't have to scroll to see alarms or the history. General Information is set as first menu item so people can instantly view it by pressing "Right Arrow" "Right Arrow".
  • Bottom bar with Overview button made smaller.
    Instead of five items, there can now be six items shown above the fold.
  • "Devices" renamed to "Products".
    This way there is a consistincy between different Victron Products, e.g. : VE PowerSetup.
